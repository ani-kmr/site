# Project Layout

Project follows the same structure:

- **Overview** → Scope, goals, and architecture  
- **Data Exploration** → Profiling, analysis, and insights  
- **Modeling** → Algorithms, schema design, and transformations  
- **Code Templates** → Reusable snippets and modules  
- **Orchestration** → Workflow automation and scheduling  
- **Deployment** → Release notes, rollout, and monitoring
